Image Gallery – CodeAlpha Internship

This project is a responsive and interactive Image Gallery web application developed as part of the CodeAlpha Internship Program.
It allows users to view images in a modern gallery layout, filter them by category, and open them in a full-screen lightbox with navigation controls.

📌 Project Overview
The Image Gallery displays images in a responsive grid layout and allows users to filter images based on categories such as Nature, Cars, and Animals.

It features:

Category filter buttons to display specific types of images
A responsive grid gallery layout
Hover animation effects on images
A lightbox view for full-screen image display
Next and previous navigation buttons in lightbox mode
Keyboard navigation support for a better user experience

This project demonstrates DOM manipulation, event handling, responsive design, and dynamic filtering using JavaScript.

🚀 Features

Image categories: Nature, Cars, Animals
Filter buttons to display images by category
Hover zoom animation on gallery images
Lightbox view for enlarged image display
Next and Previous navigation buttons
Keyboard controls (Arrow keys for navigation and Escape to close)
Responsive grid layout for different screen sizes
Modern UI with gradient background and styled buttons

🛠 Technologies Used

HTML
CSS
JavaScript

🎨 User Interface

Gradient background for a modern look
Responsive grid-based image gallery layout
Filter buttons with hover and active effects
Smooth zoom animation when hovering over images
Lightbox overlay for full-screen viewing
Navigation arrows for browsing images

⌨ Keyboard Support

Right Arrow → View next image
Left Arrow → View previous image
Escape Key → Close the lightbox

📂 How to Run

Clone or download this repository.
Open the imagegallery.html file in your web browser.
Click on any image to open it in lightbox mode.
Use the navigation arrows or keyboard keys to browse images.
Use the filter buttons to view images by category.
